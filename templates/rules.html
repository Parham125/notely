Community Rules

Last updated: 2025-10-21

These rules help keep Notely a safe and welcoming space for everyone. By using our platform, you agree to follow these guidelines.

No NSFW Content

Sexually explicit material, adult content, or inappropriate material is strictly prohibited. This includes:

  • Sexual content or explicit descriptions
  • Nudity or suggestive imagery
  • Adult-oriented discussions or material

No Advertising or Spam

Commercial advertising and spam are not allowed. This includes:

  • Unsolicited promotional content
  • Business advertisements
  • Repeated posting of similar content
  • Affiliate links without disclosure

No System Exploitation

Attempting to harm, exploit, or compromise our systems is strictly forbidden:

  • No hacking attempts or security breaches
  • No reverse engineering of our code
  • No abuse of API endpoints or rate limits
  • No creation of malicious content

Report Issues Responsibly

If you discover bugs, security vulnerabilities, or system issues, please report them responsibly:

  • Create an issue on our GitHub repository: https://github.com/Parham125/notely/issues
  • Provide detailed information about the issue
  • Do not exploit the vulnerability before reporting it
  • Allow us reasonable time to address the issue

Respectful Communication

We encourage positive and constructive interactions:

  • Treat others with respect and kindness
  • Provide helpful feedback and criticism
  • Engage in good faith discussions
  • Respect diverse opinions and backgrounds

Quality Content

Help make Notely a valuable resource:

  • Create original, thoughtful content
  • Share knowledge and experiences
  • Use appropriate tags and categorization
  • Give credit when using others' work

If you believe enforcement action was taken in error:

  • Create an issue on GitHub explaining the situation
  • Provide relevant context and evidence
  • Allow time for review and response
